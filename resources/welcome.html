👋 Welcome to the Open Web Services Inspector!

🚀 Quick Start

  1. Select or enter a "Server URL" - the server must use https
  2. Click "Send Request" to load the capabilities from the server into the UI
  3. Now try other requests such as GetMap or DescribeLayer
  4. XML and JSON output can be saved locally by clicking the Save Output button

📖 About

OWS Inspector is an open source JavaScript web application for testing Open Web Service requests (such as WMS) against geospatial servers created by Seth Girvin. The source code is on GitHub - please open any issues there. The project is written purely in JavaScript - no data is stored on this website (but requests may be logged on the servers used for testing).

This project was developed as a component of MapServer Studio. It will be fully integrated into MapServer Studio to allow dynamically editing Mapfiles to configure and test OWS services.
